# Password Locker
####
#### By ** Hope Mutheu **

## Description

This is an application that enables a user to create, delete or add a username or password using the various commands given when you run the application. This is a really fun and user friendly application.


## Setup/Installation Requirements

* Open your browser browser.
* Load gitHub on your browser.
* Search for mainamurage/triangle-tracker repository.
* Fork the data onto your own personal repository.
* To access the live website, click on this url:https://github.com/Lonzohope/passwordlocker
